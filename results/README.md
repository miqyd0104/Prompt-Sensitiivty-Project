## Table groups

- `data_quality_*`, `coverage_*`, and `missing_*`: completeness and censoring audit.
- `eda_*`: descriptive statistics, feature prevalence, and correlations.
- `rq1_*`: phrasing effects on behavioural sensitivity.
- `rq2_*`: semantic divergence, claim-risk, and recommendation-instability outcomes.
- `rq3_*`: paired model comparisons within base intents.
- `robustness_*`: common-prompt, uncapped-only, and alternative-specification analyses.
- `linguistic_influence_matrix_*`: raw, standardised, and model-specific matrix values.

`analysis_summary.json` provides machine-readable headline metadata. `RESULTS_AND_WRITEUP_GUIDE.md` provides reporting guidance and exact omnibus results. The `manual_validation/` subfolder contains a stratified coding sample and protocol for validating the automated claim-risk proxy.

`dissertation_analysis_package.zip` is a convenience archive generated by the notebook; the unpacked files beside it are the canonical repository copies.
