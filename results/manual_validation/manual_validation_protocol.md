# Manual validation protocol

1. Review the prompt and full response without looking at the automated proxy score first.
2. Mark whether a checkable factual claim is present.
3. Verify material claims using authoritative or primary sources where possible.
4. Mark support, inaccuracy/fabrication and severity using the fixed coding columns.
5. Record the verification source and a short rationale.
6. Have a second coder independently review at least 20% of the sample; report agreement (Cohen's kappa for binary fields, weighted kappa for severity).
7. Use the validated rate as the hallucination outcome. Until this is completed, call the automated measure a claim-verification risk proxy only.
